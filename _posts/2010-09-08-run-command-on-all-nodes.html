--- layout: post title: Run a command on all nodes date: '2010-09-08T12:29:00.002+01:00' author: jimbuzz tags: modified_time: '2010-09-08T12:29:48.771+01:00' blogger_id: tag:blogger.com,1999:blog-4676883399643590331.post-4565284739726301719 blogger_orig_url: http://parkyjimbo.blogspot.com/2010/09/run-command-on-all-nodes.html --- This will run the cat command on all nodes and output the results on the head node and redirect the output to a file. This gives a list of hostnames and MAC addresses in a txt file.

[root@HOST~]#cluster-fork cat /etc/sysconfig/network-scripts/ifcfg-eth0:0 | egrep "compute|HWADDR" > HostHWaddr.txt
